About Cyber Threat Intelligence Hub

Our Mission

CTIH Lite is designed to help organizations identify and classify cyber threats in real-time. Our platform uses advanced machine learning to analyze threat descriptions and provide accurate classifications with confidence scores.

How It Works

The system uses a trained machine learning model to analyze threat descriptions and classify them into various categories such as malware, phishing, DDoS attacks, and more. Each classification comes with confidence scores to help assess the accuracy of the prediction.

Security Features

  • Real-time threat classification
  • Confidence scoring for predictions
  • Blockchain-based threat history
  • Secure data storage

Technology Stack

  • Python Flask for backend
  • Scikit-learn for machine learning
  • Blockchain for data integrity
  • Modern HTML/CSS/JavaScript frontend
